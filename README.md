# Test task

## [Перейти к деплою](https://test-becloud.vercel.app/)

Реализовано приложение по получению рандомных пользователей на Vue3 + Vuetify

### Доступные Команды:
- prod сборка проекта:
```bash
npm run build
```
- запуск dev сервера:
```bash
npm run dev
```
- запуск preview:
```bash
npm run preview
```
- запуск husky:
```bash
npm run prepare
```

### Использованные библиотеки:
- Tanstack query - библиотека управления серверным состоянием
- Vuetify - UI фреймворк для Vue
- VeeValidate - библиотека управления формами для Vue
- VueRouter - библиотка для роутинга (в данном кейсе используется для записи и считывании query params)
- Axios - библиотека для запросов
